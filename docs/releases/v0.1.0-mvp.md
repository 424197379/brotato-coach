# 土豆教练 v0.1.0 MVP

这是首个公开预发布版本。

## 功能

- 商店：点击“教练建议”查看购买、锁定、属性缺口和未来规划。
- 暂停菜单：点击“分析当前局”查看当前构筑建议。
- 结算界面：点击“复盘本局”查看本局趋势和失败原因。
- 完全离线运行，不调用云端 AI，不发起网络请求。
- 使用明文 JSONL 记录低频波次/商店状态，不记录逐帧或逐次命中。

## 验证

- 两份真实黄金样本通过。
- 自动验收：35 PASS / 0 FAIL / 0 GAP。
- ZIP 结构：`mods-unpacked/BrotatoCoach-BrotatoCoach/...`

## 预发布说明

本版本尚未在当前维护者环境完成 Brotato 实机加载验证，因为构建时游戏仍在运行。安装前请：

1. 完全退出 Brotato。
2. 备份存档和 `mod_user_profiles.json`。
3. 通过 ModLoader 注册 `BrotatoCoach.zip`。
4. 首次启动后检查 `modloader.log` 中是否有 `BrotatoCoach` 错误。

遇到问题请提交 GitHub Issue，并附上相关版本和最小日志片段。
