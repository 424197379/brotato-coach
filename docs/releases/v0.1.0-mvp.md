# 土豆教练 v0.1.0 MVP

这是首个公开预发布版本。

## 功能

- 商店：点击“教练建议”查看购买、锁定、属性缺口和未来规划。
- 暂停菜单：点击“分析当前局”查看当前构筑建议。
- 结算界面：点击“复盘本局”查看本局趋势和失败原因。
- 完全离线运行，不调用云端 AI，不发起网络请求。
- 使用明文 JSONL 记录低频波次/商店状态，不记录逐帧或逐次命中。

## 验证

- 两份真实黄金样本通过。
- 自动验收：35 PASS / 0 FAIL / 0 GAP。
- ZIP 结构：`mods-unpacked/BrotatoCoach-BrotatoCoach/...`
- ZIP entry 使用正斜杠路径，已通过真实 ModLoader 启动验证。
- 维护者日志确认：zip loaded、Setup 17 mods、Initializing BrotatoCoach、三处 extension installed、
  Loaded offline coach entry points，无土豆教练错误。

## 预发布说明

本版本是 MVP 预发布。常规公开安装方案仍待 Steam 创意工坊：

- Steam 创意工坊链接：待补充

Brotato 集成版当前只枚举 Steam API 返回的已订阅创意工坊目录。本地验证时，维护者暂时将
`BrotatoCoach.zip` 与一个已订阅项目目录并列放置，再由当前 ModLoader 配置引用该 ZIP。
普通用户请等待 Steam 创意工坊版本，不建议手动改 profile。
